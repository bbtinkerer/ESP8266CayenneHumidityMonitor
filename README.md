# ESP8266 Cayenne Humidity Monitor 

By bbtinkerer (<http://bb-tinkerer.blogspot.com/>)

## Description
IOT humidity monitor using an ESP8266 module.

## Configuration

Use the Arudino IDE to upload the code to your ESP8266 module.
